# WS Team Showcase Widget

A powerful and highly customizable custom Elementor widget for showcasing team members in responsive grid or carousel layouts with advanced search, filtering, popup profiles, social links, and modern styling controls.

Built for WordPress + Elementor websites.

---

## Features

* Responsive Team Member Grid Layout
* Carousel / Slider Layout
* AJAX Powered Search & Filtering
* Popup Profile Modal
* Social Media Links
* Load More Pagination
* Fully Customizable Styling Controls
* Responsive Column Controls
* Hover Animations
* Popup Animations
* Elementor Native Controls
* Dynamic Content Support
* Lightweight Frontend Assets
* Mobile Friendly Design
* No Third-Party Dependencies

## Installation

1. Download or clone this repository.

```bash
git clone https://github.com/yourusername/ws-team-showcase-widget.git
```

2. Upload the plugin folder to:

```bash
/wp-content/plugins/
```

3. Activate the plugin from:

```text
WordPress Admin → Plugins
```

4. Ensure Elementor is installed and active.

5. Open Elementor editor and search for:

```text
WS Team Showcase
```
